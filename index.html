Exercício 3

3. Utilizar os métodos push, pop, unshift e shift para manipular um array

  1. Crie um array com o nome "frutas" e adicione 3 tipos de frutas de sua escolha
  2. Utilize o método push para adicionar uma fruta no final do array
  3. Utilize o método pop para remover a última fruta do array
  4. Utilize o método unshift para adicionar uma fruta no início do array
  5. Utilize o método shift para remover a primeira fruta do array
  6. Exiba o array resultante na tela
Resolução exercício 3
